# Install this tmux config

```sh
# install
stow  . -t ~

# uninstall
stow  -D . -t ~
```

# Tmux plugins
[GitHub - tmux-plugins/tpm: Tmux Plugin Manager](https://github.com/tmux-plugins/tpm)
```
prefix + I       : install plugins
prefix + Alt + u : uninstall plugins
```
